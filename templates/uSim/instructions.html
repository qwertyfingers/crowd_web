Starting the Simulation

Read these instructions carefully as the task will likely be new to you. If you do not submit any valid reports, your work will be rejected and you will not receive your payment!

Shown opposite is the simulation viewer. At the moment, the simulation is empty.To start the simulation click "start", the button circled in Green

The Simulation Window

A map will be loaded in the left window of the simulation viewer, and the simulation will start to run

The progress bar will show you how long the simulation has remaining. In this example, most of the simulation is remaining

Reporting on Targets

After some time, "targets" will appear (circled in green) A target is a circle that moves along the roads in the map. Your task is to write reports as quickly and accurately as possible about what it does using the menu provided.

A report is made up of 4 bits of information
1:Position
2:Size of target
3: Colour of target
4:Behaviour

1.Position

To give the position, click on the map where the target is.So if a target appears on a road, click where that happened. If a target turns a corner, click where that happened Here, the user has clicked where the target has appeared

2.Size of Target

Enter the targets size using the drop down menu. You can select between Small, Medium, and Large

3. Colour of Target

Enter the targets colour using the drop down menu. You can select between 16 different colours. Pick the one you think is closest to the targets colour.

4. Behaviour of Target

Enter the behaviour of the target by clicking one of the behaviour buttons. Here, the user has selected that the target has appeared.

Valid Reports

In order for reports to be accepted, all the information must be entered before submitting.

The example on the far left shows a good report, ready to be submitted. The 4 pieces of information have been entered

The example on the near left is not ready to be submitted. The user still has to select a position by clicking on the map. If the user trys to submit this report without adding a position, it will be rejected

Submitting Reports

Once all 4 pieces of information are entered in to a report using the menu, click the "submit the report"" button at the bottom of the menu

Once a report is submitted, the menu will be reset back to the original state

You can enter as many reports as you want in the time, but remember; for each report fill out all 4 pieces of information in the menu, then hit "submit the report"

Finishing and Sending Reports

Once the simulation has finished (the timer at the top of the menu will be completely filled), you should see a pop-up window telling you that the simulation has finished. Click ok on this window

The buttons at the bottom of the menu will have now changed to "Finish(send the reports)", shown circled in green in the image opposite

Click "Finish(send the reports)" to finish the task. If you have submitted atleast 1 valid report (containing all 4 pieces of information), you will be taken to the reward page. Follow the instructions on this page to receive your payment

If all your reports have been rejected, then you will be taken to an error page. By clicking back in your browser, you can try the simulation again.
